# Snippets Organization Plan

---

## The Common Operations

Operations

| Operation | Definition |
|---|---|
| setup | import lib, set up connections |
| head | head operation like pandas' |
| count | number of rows |
| shape | number of rows and cols |
| shapehead | shape and head |
| top | head(1).T |
| info | column info via proc content or similiar |
| col | make cols list and search for text |
| valc | simple value counts / frequency |
| run | simplest running boiler plate with no contents |
| grp | run but with group by and aggregate prefiled |
| qry | run but with full template prefiled |
| drop | drop table |
| write | create table and write |
| ctbl | drop and write and qry prefiled |
| join | join tables |
| errors | number of errors |
| log | log information |
