# Core Computer Science Subjects (Markdown Version)

## 1. Programming Fundamentals
### What it teaches
- Control flow, functions, data types  
- Modular program structure  
- Debugging and testing  
- Intro to object‑oriented programming  

### Why it matters
- Forms the base abstraction layer for all later CS subjects  
- Teaches you to express logic cleanly and predictably  
- Directly influences how you design APIs, modules, and domain objects  

### System connections
- Algorithms  
- Backend frameworks  
- Architecture patterns  

---

## 2. Data Structures & Algorithms
### What it teaches
- Lists, stacks, queues, trees, graphs  
- Sorting, searching, hashing  
- Time/space complexity  
- Algorithmic problem‑solving  

### Why it matters
- Enables efficient data handling in real systems  
- Critical for performance, scaling, and predictable behaviour  

### System connections
- API performance  
- Database indexing  
- Caching and pagination  

---

## 3. Discrete Mathematics & Logic
### What it teaches
- Sets, functions, relations  
- Logic and proofs  
- Boolean algebra  
- Graph theory  

### Why it matters
- Provides the formal reasoning tools behind all software correctness  
- Helps you design stable, invariant‑preserving systems  

### System connections
- Database schema design  
- Routing graphs  
- State machines  

---

## 4. Databases
### What it teaches
- SQL (joins, indexes, transactions)  
- Normalisation  
- NoSQL models  
- Query optimisation  

### Why it matters
- Every app is data‑driven  
- Good schema design prevents scaling failures  

### System connections
- API design  
- Caching layers  
- Authentication/session storage  

---

## 5. Operating Systems
### What it teaches
- Processes, threads, scheduling  
- Memory management  
- File systems  
- Concurrency and synchronization  

### Why it matters
- Modern apps rely heavily on concurrency  
- Helps you reason about performance and race conditions  

### System connections
- Node.js event loop  
- Thread pools  
- Mobile app lifecycle  

---

## 6. Computer Networks
### What it teaches
- TCP/IP, HTTP, DNS  
- Routing and switching  
- Latency, throughput, congestion  
- Network security basics  

### Why it matters
- Web and mobile apps are networked systems  
- Understanding the transport layer improves API design  

### System connections
- REST/GraphQL performance  
- CDN behaviour  
- WebSockets and real‑time systems  

---

## 7. Software Engineering
### What it teaches
- Requirements gathering  
- System design  
- Testing strategies  
- Version control  
- Agile methodologies  

### Why it matters
- Teaches you to build production‑grade systems  
- Bridges theory and real‑world engineering  

### System connections
- Architecture patterns  
- CI/CD pipelines  
- Code quality and maintainability  

---

## 8. Cybersecurity Fundamentals
### What it teaches
- Threat models  
- Encryption and hashing  
- Authentication and authorization  
- Secure coding practices  

### Why it matters
- Web/mobile apps are the primary attack surface  
- Security failures are catastrophic  

### System connections
- JWT, OAuth  
- SQL injection prevention  
- Secure session management  

---

## 9. Human–Computer Interaction (HCI)
### What it teaches
- UX principles  
- Usability testing  
- Interface design  
- Accessibility  

### Why it matters
- Apps succeed or fail based on UX  
- Gives you the vocabulary to design intuitive interfaces  

### System connections
- Mobile UI patterns  
- Component libraries  
- User flows and onboarding
