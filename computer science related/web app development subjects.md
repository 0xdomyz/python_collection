# Web & App Development Specialised Subjects

## 1. Front-End Web Development
### What it teaches
- HTML, CSS, JavaScript fundamentals  
- Modern frameworks (React, Vue, Angular)  
- Component-based UI architecture  
- Responsive design and accessibility  

### Why it matters
- Front-end is the user-facing layer of most apps  
- Teaches how to build maintainable, modular interfaces  
- Forms the foundation for full-stack development  

### System connections
- API consumption  
- State management  
- UI component trees and rendering pipelines  

---

## 2. Back-End Web Development
### What it teaches
- Server-side frameworks (Node.js, Django, Spring, .NET)  
- REST APIs and GraphQL  
- Authentication and authorization  
- Middleware, routing, and request handling  

### Why it matters
- Back-end logic powers all business rules and data flow  
- Teaches how to design stable, scalable services  

### System connections
- Databases  
- Caching layers  
- Cloud deployment  

---

## 3. Full-Stack Engineering
### What it teaches
- Integrating front-end and back-end systems  
- API design and consumption  
- Session management and cookies  
- Deployment pipelines and CI/CD  

### Why it matters
- Full-stack skills make you versatile and employable  
- Helps you understand the entire request lifecycle  

### System connections
- DevOps  
- Containerisation  
- Distributed systems  

---

## 4. Mobile App Development (iOS/Android)
### What it teaches
- Native iOS (Swift) and Android (Kotlin/Java)  
- Mobile UI/UX patterns  
- App lifecycle and state management  
- Accessing device sensors and APIs  

### Why it matters
- Mobile-first is the dominant user platform  
- Teaches constraints like battery, memory, and offline mode  

### System connections
- REST/GraphQL APIs  
- Push notifications  
- Local storage and caching  

---

## 5. Cross-Platform App Development
### What it teaches
- Flutter, React Native, or similar frameworks  
- Shared UI logic across platforms  
- Platform bridges and native modules  
- Performance considerations  

### Why it matters
- Reduces development cost and time  
- Teaches abstraction trade-offs between native and hybrid  

### System connections
- Component libraries  
- API integration  
- Build pipelines  

---

## 6. Cloud & Deployment
### What it teaches
- AWS, Azure, or GCP basics  
- Serverless functions  
- Containers (Docker)  
- CI/CD pipelines  

### Why it matters
- Modern apps run in the cloud  
- Teaches how to deploy, scale, and monitor applications  

### System connections
- Load balancing  
- Logging and observability  
- Infrastructure as code  

---

## 7. Web Security
### What it teaches
- OWASP Top 10  
- Secure authentication (OAuth, JWT)  
- Input validation and sanitisation  
- HTTPS, CORS, CSRF, XSS  

### Why it matters
- Web apps are the most attacked systems  
- Security failures are expensive and reputation-damaging  

### System connections
- API gateways  
- Identity providers  
- Secure session management  

---

## 8. UX/UI Design for Web & Mobile
### What it teaches
- Wireframing and prototyping  
- Design systems and component libraries  
- Usability testing  
- Accessibility standards  

### Why it matters
- Good UX drives user retention  
- Helps developers collaborate with designers  

### System connections
- Front-end frameworks  
- Mobile navigation patterns  
- User onboarding flows  

---

## 9. Software Engineering Project / Capstone
### What it teaches
- Building a full production-grade app  
- Working in teams  
- Requirements gathering and documentation  
- Testing, deployment, and maintenance  

### Why it matters
- Simulates real industry workflows  
- Produces portfolio-ready work  

### System connections
- Everything: front-end, back-end, mobile, cloud, security
